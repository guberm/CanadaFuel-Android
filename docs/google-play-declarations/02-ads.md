# Ads Declaration

Category: App content > Ads

## Recommended declaration
- Does your app contain ads: Yes

## Evidence in app
- AdMob app id configured in Android manifest.
- Banner ad loaded and shown in home screen bottom area.

## Action checklist
1. Set Contains ads to Yes in Play Console.
2. Ensure this matches in-app behavior for all release variants.
3. Keep declaration updated if ads are removed later.
