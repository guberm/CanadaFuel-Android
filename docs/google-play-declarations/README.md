# Google Play Declarations Pack

This folder contains one markdown file per required Play Console app-content declaration category.

## Files
- 01-privacy-policy.md
- 02-ads.md
- 03-app-access.md
- 04-content-ratings.md
- 05-target-audience-and-content.md
- 06-data-safety.md
- 07-advertising-id.md
- 08-government-apps.md
- 09-financial-features.md
- 10-health-apps.md

## Usage
1. Open each file and copy the recommended values into Play Console.
2. Replace placeholders where needed (especially privacy policy public URL).
3. Keep this folder updated when app behavior changes.
