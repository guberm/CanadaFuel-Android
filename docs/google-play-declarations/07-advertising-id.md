# Advertising ID Declaration

Category: App content > Advertising ID

## Recommended declaration
- Does your app use Advertising ID: Yes

## Rationale
Google Mobile Ads SDK is integrated for banner ads, which uses Advertising ID for ad-related functions.

## Action checklist
1. Select Yes for Advertising ID usage.
2. Confirm your app usage aligns with Play policy and privacy disclosures.
3. Ensure targetSdk releases are blocked until this declaration is complete.
